### 🌊 Water Turbidity Monitoring System

#### 📌 Overview

The Water Turbidity Monitoring System is an Arduino-based embedded project designed to monitor water quality in real time.

It automatically controls a water pump based on turbidity levels and odor detection, ensuring safe and efficient water management through intelligent automation.

The system continuously evaluates water conditions using multiple sensors and triggers safety actions whenever abnormal readings are detected.

---
#### 🧰 Tools & Components Used

#### 🔄 System Workflow

- 🌊 Turbidity sensor → measures water clarity  
- 🧪 Gas sensor → detects odor / contamination  
- 🧠 Arduino → processes data in real time  
- ⚡ Relay module → controls water pump  
- 🚨 Safety system → stops pump on bad water quality  
- 📟 LED display → shows live status & readings  

---
#### ⭐ Key Features

- 🌊 Real-time monitoring of water quality  
- 🧪 Gas sensor-based odor detection  
- ⚙️ Automated pump control system  
- 🚨 Safety mechanism for automatic shutdown  
- 📟 Live system status display  
- 🤖 Embedded automation using Arduino
